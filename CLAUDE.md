This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Commands

```bash
pnpm dev          # Start dev server at localhost:4321
pnpm build        # Build production site to ./dist/
pnpm preview      # Preview production build locally
pnpm astro check  # Type-check the project
```

## Architecture

This is an **Astro + Starlight** documentation site using the `starlight-theme-galaxy` theme.

- **Content lives in `src/content/docs/`** as `.md` or `.mdx` files. File paths map directly to URL routes.
- **Sidebar navigation** is configured in `astro.config.mjs` — either manually via `items` or auto-generated via `autogenerate: { directory: '...' }`.
- **Images** go in `src/assets/` and are referenced with relative paths from Markdown. Static assets (favicons, etc.) go in `public/`.
- **Content collection schema** is defined in `src/content.config.ts` using Starlight's built-in `docsSchema`.

Adding a new doc page: create a `.md` or `.mdx` file under `src/content/docs/` and (if needed) add it to the sidebar in `astro.config.mjs`.
