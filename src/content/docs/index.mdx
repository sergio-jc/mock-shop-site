## Mock Shop API

API pública para simular un e-commerce. Accedé a productos, órdenes e inventario
tanto por REST como por GraphQL, sin autenticación para endpoints de lectura.

<UrlBox label="Base API URL" url={BASE_API_URL} />

---

## API REST

<UrlBox label="REST API URL" url={`${BASE_API_URL}/v1`} />

Endpoints RESTful con respuestas en JSON. Los endpoints de lectura son públicos;
las operaciones de escritura requieren un token Bearer en el header `Authorization`.

---

## GraphQL

<UrlBox label="GraphQL API URL" url={`${BASE_API_URL}/graphql`} />

Endpoint único. Enviá peticiones `POST` con un body JSON que contenga tu query.
Explorá el schema completo en el playground integrado.
